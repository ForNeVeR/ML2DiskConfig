Настройка диска ML2

(версия 1.6.3)

История версий

Обозначения:
- Изменено.
- Добавлено.

1.6.3
-Теперь в окне настроек есть кнопки "ОК" и "Отмена".

1.6.2.1
-Упорядочено чередование по нажатию клавиши Tab элементов окна настроек.

1.6.2
-Немного изменен дизайн некоторых окон программы.
-Добавлен параметр "Быстрое применение изменений", позволяющее не нажимать кнопку "Сохранить" вкладки "Вручную" для применения изменений.

1.6.1.1
-Исправлен глюк: при нажатии кнопки "Отмена" в окне выбора значка появлялось сообщение "Не найден файл значка".
-При удалении файла autorun.inf соответствующие элементы окна программы становятся неактивными.

1.6.1
-Можно изменить вкладку, активную после запуска программы.
-Изменена функция определения первого запуска: теперь в случае, если программа была скопирована на другой компьютер (или система была переустановлена), определится первый запуск.
-Немного изменено окно "О программе...".

1.6
-Программа позволяет изменять метку диска.
-Программа может работать в режиме ограниченной функциональности даже без создания файла autorun.inf.

1.5.2
-Улучшено управление интеграцией в систему: теперь программа сообщает, обнаружена интеграция или нет и позволяет только соответствующие действия (теперь, например, нельзя убрать интеграцию, которой не было). Программа также сообщает, если интеграция повреждена (если, например, файл программы был перемещён).
-Интеграция в систему теперь корректно убирается при деинсталляции.
-Переработана справка программы.
-В инсталляторе появились новые опции: просмотр справки и интеграция программы в систему сразу после инсталляции.

1.5.1
-Все элементы главного окна программы переключаются по нажатию клавиши Tab упорядоченно.
-Изменено расположение некоторых кнопок.
-После того, как пользователь дал указание создать файл autorun.inf на диске, этот диск активируется для работы программы.
-Теперь в истории версий есть картинки.

1.5
-Можно конкретно указать диски, предназначенные для настройки.

1.4.1
-Можно указать диски, которые не нужно проверять при запуске программы.

1.4.0.1
-При закрытии программы через системный "крестик" все настройки теперь сохраняются.
-Переделано окно "О программе...".
-Файл ReadMe назван справкой и перенесён в формат chm.

1.4
-В случае назначения значка не с настраиваемого диска он копируется на настраиваемый с именем icon.ico.

1.3.3
-Можно отменить проверку дисков A: и B:.

1.3.2
-Программа теперь занимает меньше места в оперативной памяти.
-Оптимизирован код, исправлены мелкие недоработки.

1.3.1
-Перед созданием файла autorun.inf (если его не существует) задаётся вопрос.
-Если у диска не настроен значок, то в поле просмотра значка загружается пустой (раньше там мог сохраниться значок предыдущего диска).
-Мелкие изменения в интерфейсе и работе программы.
-В заголовке теперь написано, какой диск настраивается.

1.3
-По многочисленным заявкам тестеров теперь вместо ввода буквы диска её нужно выбирать из списка.
-Эта версия не требует никаких библиотек!
-Создана программа установки.

1.2
-Исправлены найденные глюки (в основном, связанные с выбором программы автозапуска).
-Программа может использовать значки из .exe и .dll. При их использовании программа копирует выбранный значок (не весь файл!) в корневой каталог диска с именем icon.ico и применяет его.
-Программа интегрируется в оболочку.
-Перед удалением файла autorun.inf задаётся соответствующий вопрос.
-Произведены мелкие изменения в интерфейсе.
-Файл ReadMe переведён в HTML.

1.1
-В исполняемый файл программы добавлена некоторая информация о версии.
-Теперь программа корректно записывает информацию о значке и файле автозапуска, если они находятся на другом диске.
-Программа корректно читает файл ReadMe.

1.0
Первая версия программы. Как показали последующие версии, самая глючная.
