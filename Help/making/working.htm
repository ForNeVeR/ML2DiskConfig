Настройка диска ML2

(версия 1.6.3)

Начало работы

Если данный запуск программы является первым после установки программы, то она спросит, какие диски подлежат настройке. Нужно ввести буквы дисков без пробелов и разделителей, например "CDFG". Программа позволяет настраивать только локальные диски (и, возможно, сетевые). CD, дискеты и т.д. настраивать нельзя.

Для начала нужно выбрать диск, с которым будут осуществляться дальнейшие действия. Для этого нужно выбрать букву этого диска из выпадающего списка "Настраиваемый диск" и нажать кнопку "ОК". Если на диске уже есть файл "autorun.inf", то программа считает данные из него. Если же такого файа на диске нет, то программа предложит его создать (иначе программа будет работать в режиме ограниченной функциональности - будут доступны только средства вкладки "Параметры диска").

После выбора диска можно либо вручную отредактировать текст файла autorun.inf (если вы, конечно, в этом разбираетесь), выбрав вкладку "Вручную", либо воспользоваться средствами вкладки "Автоматически", которая выбрана по умолчанию после запуска программы. Также можно настроить некоторые общие параметры диска на соответствующей вкладке.

Вкладка "Автоматически"

На данной вкладке находятся 3 раздела: "Значок", "Автоматически запускаемый файл" и "Команды контекстного меню" и 2 кнопки снизу: "Применить изменения" и "Удалить файл autorun.inf".

Раздел "Значок" показывает значок, выбранный для диска сейчас, имя файла, содержащего значок и кнопку "Изменить...". При нажатии последней появляется стандартное диалоговое окно открытия файла. Выберите файл, с содержащий нужный вам значок и нажмите кнопку "Открыть". Если этот файл - файл формата ico, то он немедленно будет применён к диску. Если файл находится не на настраиваемом диске, то он будет скопирован в корневой каталог этого диска с именем icon.ico. Вам нужно учесть три вещи. Во-первых, если файл с этим именем существует и каким-либо образом защищён от записи, то полезут ошибки. Во-вторых, если он не защищён, то будет перезаписан. И в-третьих, программа не всегда корректно проводит эту операцию (могут потеряться цвета, прозрачность и т.д.). Так что лучше позаботиться о том, чтобы файл был на настраиваемом диске (не важно, в каком каталоге), заранее.
Если выбранный файл - файл формата dll или exe, то появится окно выбора значка.
Чтобы выбрать значок, нужно сначала навести на него "прицел" с помощью кнопок "<<" и ">>", а заетм нажать кнопку "Выбрать". Выбранный значок будет сохранён на настраиваемый диск с именем icon.ico. Вам нужно учесть те же три вещи. Во-первых, если файл с этим именем существует и каким-либо образом защищён от записи, то полезут ошибки. Во-вторых, если он не защищён, то будет перезаписан. И в-третьих, программа не всегда корректно проводит эту операцию (могут потеряться цвета, прозрачность и т.д.). Так что лучше позаботиться о том, чтобы значок был на настраиваемом диске (есть много программ, позволяющих корректно "вытянуть" значок, например, Resource Hacker).

Раздел "Автоматически запускаемый файл" позволяет выбрать файл, запускаемый автоматически после двойного щелчка по диску. Можно, например, поставить программу, которая выводит сообщение "НЕ ЛЕЗЬ!!!". Обойти автозапуск можно, щёлкнув по диску правой кнопкой и нажав "Открыть". В разделе показывается имя автоматически запускаемого файла и есть кнопка "Изменить...". После щелчка по ней выведется, опять же, стандартное окно открытия файла. Учтите, выбранный файл должен находиться на настраиваемом диске! Программа не будет его копировать из соображений целостности (ведь этот файл может требовать другие файлы).

Раздел "Команды контекстного меню" позволяет ввести допонительные команды контекстного меню диска. Это пункты, выводящиеся в меню при правом щелчке по диску. Раздел содержит текстовое поле, содержащее исходный текст команд и кнопку "Добавить...". При щелчке по ней появится окно "Настройка команд контекстного меню".
Нужно ввести в поле "Имя пункта меню" любое сочетание английских букв (оно фигурирует только в файле autorun.inf), в поле "Текст пункта меню" само название пункта, которое появится потом в меню, в поле "Выполняемая команда" саму команду. Также есть кнопка "Обзор...", позволяющая выбрать файл, который будет запускаться при выборе пункта. После нажатия кнопки "Добавить" исходный текст пункта появится в поле для исходного текста. Окно при этом не закроется, что удобно при добавлении нескольких пунктов подряд. Исходный текст автоматически будет скопирован куда нужно при применении изменений. Для удаления пункта нужно удалить его исходный текст.

Также на вкладке "Автоматически" есть две кнопки, не относящихся ни к одному из разделов: "Применить изменения" и "Удалить файл autorun.inf".
О кнопке "Применить параметры" сказано ниже.
При нажатии на кнопку "Удалить файл autorun.inf" файл autorun.inf с настраиваемого диска будет удалён и все настройки диска, кроме тех, которые были внесены на вкладке "Параметры диска", будут стёрты. Учтите, что после удалении файла autorun.inf нужно заново выбрать диск для настройки (если вы собираетесь ещё что-либо настраивать).

Вкладка "Вручную"

Вкаладка "Вручную" содержит исходный текст файла autorun.inf. Она позволяет отредактировать его вручную. Содержит две кнопки: "Обновить" и "Сохранить". При нажатии на кнопку "Обновить" текст заново считывается из файла autorun.inf. Это полезно, например, при изменении текста в другом редакторе во время работы программы. Кнопка "Сохранить" позволяет сохранить файл autorun.inf на диск в таком виде, в каком он находится в текстовом поле.

Применение параметров

После нажатия кнопки "Применить изменения" вкладки "Автоматически" дальнейшие действия будут зависить от того, активирована ли опция "Быстрое применение изменений". Если активирована, то изменения немедленно будут применены, а если не активирована, то программа выведет вкладку "Вручную", где пользователь должен нажать кнопку "Сохранить".

Вкладка "Параметры диска"

На этой вкладке пока есть только поле "Метка" и кнопка для смены этой метки. Обратите внимание, что сменить метку на пустую пока нельзя.

Настройка программы

Поле нажатия кнопки "Настройки..." появится окно настроек.
Пункт "Проверять диски" позволяет назначить диски, предназначенные для настройки. Только их можно будет настроить с помощью программы. Рекомендуется ввести сюда все локальные диски компьютера. Вводить буквы дисков нужно без пробелов и разделителей, как показано на рисунке.

Пункт "Активная вкладка" позволяет определить вкладку, активную сразу после запуска программы.

Пункт "Быстрое применение изменений" позволяет применять изменения к диску по нажатию кнопку "Применить изменения" вкладки "Автоматически", без последующего нажатия кнопки "Сохранить" вкладки "Вручную".

Можно интегрировать программу в оболочку или отменить интеграцию при помощи соответствующих кнопок. Интеграция в систему позволит вызывать Настройку диска ML2 из контекстного меню любого диска. В случае, если в строке "Статус" написано "Интеграция повреждена", её рекомендуется добавить заново.

После того, как нужные настройки выполнены, надо нажать кнопку "ОК" для применения изменений.
