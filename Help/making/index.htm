Настройка диска ML2

(версия 1.6.3)

Описание

Программа предназначена для настройки диска путём изменения файла autoexec.inf (при этом никаких специальных знаний не требуется). Она обладает удобным интерфейсом и поддерживает стили Windows XP и выше. Никаких вредных для системы действий не производит. Запрашивает подтверждения на потенциально опасные действия во избежание потери информации.
Перед установкой новой версии рекомендуется (хотя и необязательно) удалить предыдущую.

Программа распространяется бесплатно по принципу "как есть", авторы программы не несут никакой ответственности за нанесение вреда вам и вашему компьютеру при помощи данной программы.

Возможности программы

Программа позволяет осуществлять следующие действия:

Действия с дисками:

  • Изменение метки диска
  • Добавление диску значка
  • Добавление программы автозапуска
  • Добавление дополнительных команд контекстного меню
  • Ручное редактирование файла autorun.inf
Служебные действия:
  • Интеграция в систему
  • Добывание значков из .exe, .dll файлов.

Требования к системе

Для работы программы Настройка диска ML2 последней версии необходимо:
  • Microsoft Windows 95, 98, 98SE, NT, 2000, XP, 2003 Server, Longhorn
  • До 900 кб места на диске

Обратите внимание

Программа тестировалась на Windows XP. На остальных операционных системах (в особенности на Windows 2000) мы не может обещать работу программы, однако, теоретически, она должна работать и там.

При смене метки диска её нельзя менять на пустую. В случае, если поле оставлено пустым, метка сменена не будет.

Обнаружено, что программа не всегда корректно "вытягивает" значки из exe и dll (особенно относится к значкам палитры XP с альфа-каналом)! В будущих версиях это, по возможности, будет исправлено.
